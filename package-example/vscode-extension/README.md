# MAAF Core Snippets

VS Code code snippets for MAAF Core framework.

## Features

- ✅ Application bootstrap snippet
- ✅ Controller template snippet
- ✅ Module template snippet
- ✅ Middleware template snippet

## Installation

Install from VS Code Marketplace or manually:

```bash
code --install-extension maaf-core-snippets-1.0.0.vsix
```

## Usage

Type the snippet prefix and press Tab:

- `maaf-app` - Application bootstrap
- `maaf-controller` - Controller template
- `maaf-module` - Module template
- `maaf-middleware` - Middleware template
